#FruitCrushGame MVC
*This project is a web game developed using ASP.NET MVC.
*Players match the same type of fruits in rows or columns of three or more to score points.
*The game includes different levels, limited moves, timed levels, and special features like ice blocks.
#How to Play
Each level has a different goal and board size.
Difficulty increases from Level 1 to Level 5:
Some levels have a limited number of moves.
Some levels are timed.
Some levels contain ice blocks that must be broken.
Match fruits to earn points, and reach the level goal to advance to the next level.
Upon completing Level 5, a “GAME COMPLETED” message appears with a confetti animation.
#Features
Multiple levels with increasing difficulty
Ice blocks and target scores
CRUSH animation
Confetti effect and end-of-game message
Responsive and colorful game interface
#Technologies
HTML & Razor (ASP.NET Core)
CSS (Flexbox & Grid)
JavaScript
